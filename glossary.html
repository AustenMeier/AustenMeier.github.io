Glossary

  • HTML Elements
  • Attributes
  • Audio/Video DOM
  • HTTP Messages
  • Keyboard Shortcuts

HTML Elements

HTML elements are the building blocks of HTML pages. They represent structure and content in a web page.

Attributes

HTML attributes provide additional information about HTML elements. They are always included in the opening tag of an element and usually come in name/value pairs like name="value".

Audio/Video DOM

The Audio and Video DOM allows you to manipulate audio and video elements in your web pages, providing methods, properties, and events for control.

HTTP Messages

HTTP messages are how data is exchanged between a server and a client. They include requests made by a client and responses from the server.

Keyboard Shortcuts

Keyboard shortcuts are combinations of keys that provide quick access to certain functions within an application or operating system.
